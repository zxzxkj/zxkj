说明
    本脚本是图色脚本，基于python标准计算机视觉识别实现，非外挂。如有侵犯某方合法权益，请来函说明，我将配合作出修改。
    为上班族或搬砖党等多开玩家提供一个可靠的脚本，以便大家可以从枯燥的重复活动中解放出来，有精力从事更多有趣味或是更有性价比的工作。
    标准python程序，不对计算机系统进行任何修改，也没有任何采集计算机敏感数据或上传数据功能。不放心的兄弟可以自行分析测试。

功能 
    自动登录游戏： 需在配置文件中设置你的游戏路径、区服名、角色名
    无需设置的：
    刮刮乐、帮派签到、买药（气血储备低于500000万会自动进行）、领取活跃奖励、运镖、选举乡试、三界奇缘、趣闻鉴赏，普通副本、宝图任务、挖宝图任务
    需要设置的：
    捉鬼任务次数自行设置 是否执行东海封妖自行设置 是否执行自动烹饪或自动炼药 自动上架 自动卖物 丢垃圾

使用方法：
    默认配置文件中设置好你的游戏路径、区服名、角色名，填入注册码就可以使用了。别的看自己使用情况修改。
